# SynapseFlow - AI-Powered Productivity Platform

A modern, fully responsive landing page for a fictional AI-powered productivity tool called SynapseFlow. Built with pure HTML and CSS - no JavaScript required!


#  Features

路 Pure CSS Implementation: No JavaScript dependencies
路 Fully Responsive: Mobile-first design with media queries
路 Modern UI/UX: Clean, professional design with smooth animations
路 CSS-Only Interactions: Hover effects, transitions, and carousel
路 Accessible: Semantic HTML and proper ARIA labels
路 Cross-Browser Compatible: Works on all modern browsers


#  Technologies Used

路 HTML5: Semantic markup
路 CSS3:
  路 CSS Grid & Flexbox
  路 CSS Variables
  路 Animations & Transitions
  路 Media Queries
  路 Pseudo-classes (:hover, :checked)

  #  Key CSS Techniques

路 CSS Grid & Flexbox: For responsive layouts
路 CSS Variables: For consistent theming
路 CSS Animations: For interactive elements
路 Media Queries: For responsive design
路 Pseudo-classes: For interactive states
路 CSS-only Carousel: Using radio inputs and sibling selectors

#  Project Purpose

This project demonstrates:
